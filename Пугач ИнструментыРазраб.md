**1.Инструменты разработки**

Инструментами разработки будут являться:

- операционная система Windows 10;

- язык программирования C#;

- фреймворк ASP.NET Core 8;

- среда разработки Visual Studio 2022 Community / Visual Studio Code;

- система управления базами данных Microsoft SQL Server 2022;

- система контроля версий Git;

- программное средство для создания UML диаграмм Draw.io / StarUML;

- пакет программ Microsoft Office 2021 для оформления документации.

Данное программное средство будет разрабатываться в среде программирования Visual Studio 2022 Community, которая предоставляет мощные инструменты для разработки веб-приложений на платформе .NET, включая встроенную поддержку ASP.NET Core, отладку, профилирование и интеграцию с системами контроля версий. Visual Studio 2022 является современной IDE, обеспечивающей высокую производительность разработчика и удобство работы с кодом на C#.

В качестве операционной системы для разработки может использоваться Windows 10, как наиболее распространенная и удобная среда для работы с Visual Studio.

Язык программирования C# – объектно-ориентированный язык программирования, разработанный компанией Microsoft. C# является основным языком для платформы .NET и ASP.NET Core. Он поддерживает современные парадигмы программирования: строгую типизацию, полиморфизм, обобщенные типы, асинхронное программирование, LINQ, а также множество других возможностей, необходимых для создания надежных и производительных веб-приложений. В проекте будет использоваться C# версии 12 (в составе .NET 8).

Фреймворк ASP.NET Core 8 — это кроссплатформенная высокопроизводительная среда с открытым исходным кодом для создания современных веб-приложений и API. ASP.NET Core обеспечивает гибкую архитектуру, встроенную поддержку внедрения зависимостей, middleware, аутентификацию, а также удобную работу с базами данных через Entity Framework Core. Для реализации чата в реальном времени будет использоваться библиотека SignalR, входящая в состав ASP.NET Core.

Visual Studio Code — легкий, но мощный редактор кода, который может использоваться как альтернатива Visual Studio, особенно при разработке на Linux. Поддерживает расширения для C#, отладку, интеграцию с Git и терминалом.

В качестве системы управления базами данных выбрана Microsoft SQL Server 2022 – надежная и производительная реляционная СУБД, поддерживающая все необходимые возможности для хранения и обработки данных туристического каталога. Для взаимодействия с БД будет использоваться Entity Framework Core (ORM), что позволит работать с данными через объекты C# и автоматически генерировать SQL-запросы.

Система контроля версий Git необходима для управления исходным кодом, совместной работы и отслеживания изменений. Репозиторий может размещаться на GitHub или GitLab.

Для создания UML-диаграмм (вариантов использования, классов, последовательностей и т.д.) будут применяться инструменты Draw.io. Эти средства позволяют создавать диаграммы в соответствии с нотацией UML и экспортировать их в графические форматы для включения в документацию.

Пакет Microsoft Office 2021 используется для оформления текстовой документации (технического задания, пояснительной записки, руководств) и создания отчетов.
