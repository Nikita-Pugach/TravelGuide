# Business Domain — TravelGuide

**Purpose**: Бизнес-контекст проекта TravelGuide — справочник туриста.

## Project Identity

```
Project Name: TravelGuide — Справочник туриста
Tagline: Каталог туров, отелей и достопримечательностей
Problem Statement: Нужна удобная платформа для поиска и бронирования туров
Solution: Веб-приложение с каталогом, фильтрами, отзывами, бронированием и чатом
```

## Target Users

| User Segment | Who They Are | What They Need | Pain Points |
|--------------|--------------|----------------|-------------|
| Туристы | Люди, ищущие туры | Каталог, фильтры, отзывы, бронирование | Сложно сравнить варианты |
| Менеджеры | Сотрудники агентств | Управление турами, обработка бронирований | Ручная работа с заявками |
| Админы | Владельцы платформы | Полный контроль контента и пользователей | Нет единой панели управления |

## Value Proposition

- **Для туристов**: удобный поиск с фильтрами, карты, отзывы, бронирование онлайн
- **Для менеджеров**: обработка бронирований, чат поддержки, статистика
- **Для админов**: CRUD всего контента, модерация, Excel-отчёты

## Key Features

1. Каталог туров/отелей/достопримечательностей с фильтрами и пагинацией
2. Страницы деталей с Leaflet-картами
3. Отзывы с модерацией
4. Бронирование туров
5. Чат поддержки (SignalR)
6. 5 типов Excel-отчётов
7. Избранное
